Documentacion

Diferenciación Numérica

La diferenciación numérica es una técnica utilizada para aproximar la derivada de una función en un punto cuando no se dispone de la expresión analítica de la función o cuando es difícil de derivar. Se basa en el uso de valores de la función en puntos cercanos.

El concepto fundamental se deriva de la definición de derivada:

f'(x) = limh→0 (f(x+h) - f(x)) / h

En la práctica, no podemos tomar h infinitamente pequeño, por lo que usamos valores pequeños de h para aproximar la derivada.

Tipos de fórmulas de diferenciación numérica:

  • Diferencias hacia adelante
  • Diferencias hacia atrás
  • Diferencias centradas

Fórmulas de Diferenciación Numérica

1. Diferencias hacia adelante

f'(x) ≈ (f(x+h) - f(x)) / h

Error: O(h)

2. Diferencias hacia atrás

f'(x) ≈ (f(x) - f(x-h)) / h

Error: O(h)

3. Diferencias centradas

f'(x) ≈ (f(x+h) - f(x-h)) / (2h)

Error: O(h²) - Más preciso que las anteriores

Fórmulas de mayor orden

Para la segunda derivada:

f''(x) ≈ (f(x+h) - 2f(x) + f(x-h)) / h²

Error: O(h²)

Diferencias Divididas

Las diferencias divididas son una herramienta fundamental en el cálculo de polinomios de interpolación, especialmente en la forma de Newton. También se pueden usar para aproximar derivadas.

Definición recursiva:

f[xi] = f(xi)
f[xi, xi+1, ..., xi+k] = (f[xi+1, ..., xi+k] - f[xi, ..., xi+k-1]) / (xi+k - xi)

Tabla de diferencias divididas:

Se organizan en una tabla triangular donde cada nivel representa un orden mayor de diferencias:

x f(x) Primera diferencia Segunda diferencia Tercera diferencia
x0 f[x0] f[x0, x1] f[x0, x1, x2] f[x0, x1, x2, x3]
x1 f[x1] f[x1, x2] f[x1, x2, x3]
x2 f[x2] f[x2, x3]
x3 f[x3]

Relación con derivadas:

Cuando los puntos están igualmente espaciados y h → 0, las diferencias divididas convergen a las derivadas:

f'(x) ≈ f[x, x+h] = (f(x+h) - f(x)) / h

Implementación en Python

Python 
# Diferenciación numérica
def forward_diff(f, x, h):
    return (f(x+h) - f(x)) / h

def backward_diff(f, x, h):
    return (f(x) - f(x-h)) / h

def centered_diff(f, x, h):
    return (f(x+h) - f(x-h)) / (2*h)

# Diferencias divididas
def divided_diff(x, y):
    n = len(y)
    coef = y.copy()
    for j in range(1, n):
        for i in range(n-1, j-1, -1):
            coef[i] = (coef[i] - coef[i-1]) / (x[i] - x[i-j])
    return coef

# Ejemplo de uso
x_points = [1, 2, 4, 5]
y_points = [2, 3, 11, 20]
print("Coeficientes:", divided_diff(x_points, y_points))
