Documentacion

Para el desarrollo del proyecto se utilizó Python como lenguaje principal por su facilidad de uso y potencia. Se integró NumPy para realizar cálculos numéricos eficientes y Matplotlib para generar gráficos y visualizar datos de forma clara. Sympy permitió resolver operaciones matemáticas simbólicas como derivadas e integrales. Finalmente, se utilizó el framework Django para crear la estructura web del sistema, facilitando la gestión del backend y la interfaz de usuario.


Python

Es un lenguaje de programación versátil ampliamente utilizado en el análisis numérico por su simplicidad y poder. Permite implementar métodos numéricos de forma clara y eficiente, y es compatible con bibliotecas especializadas para cálculos matemáticos, visualización y resolución de problemas científicos.

NumPy

Es una biblioteca fundamental para el análisis numérico en Python. Proporciona estructuras de datos eficientes, como los arreglos multidimensionales (ndarray), y funciones matemáticas de alto rendimiento. Es ideal para operaciones vectorizadas, álgebra lineal, interpolación y otros métodos numéricos usados en problemas científicos y de ingeniería.

Instalación

Windows
Debian


Matplotlib

Es una biblioteca de Python utilizada para la creación de gráficos y visualizaciones. En análisis numérico, permite representar funciones, datos experimentales y resultados de métodos numéricos como aproximaciones, soluciones de ecuaciones o errores, facilitando la interpretación visual de los resultados.

Instalación

Windows
Debian


Sympy

Es una biblioteca de Python para matemáticas simbólicas. Permite realizar operaciones como derivadas, integrales, simplificaciones y resolución de ecuaciones de forma exacta. Aunque se enfoca en el álgebra simbólica, es útil en análisis numérico para validar resultados analíticos y generar expresiones que luego se pueden evaluar numéricamente.

Instalación

Debian
